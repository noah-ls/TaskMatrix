# Changelog

All notable changes to TaskMatrix are documented in this file.

The project uses version tags in the form `vMAJOR.MINOR.PATCH`.

## Unreleased

### Added

- Release process documentation.
- Tag-driven GitHub Release workflow for building, testing, packaging, and
  uploading a DMG.
- Reusable DMG packaging script.
- Expanded privacy policy.

## 1.0.0 - 2026-07-23

### Added

- Native macOS Eisenhower Matrix task board.
- Task create, edit, delete, complete, move, and reorder workflows.
- Task pinning.
- Subtasks.
- Due dates with date-only formatting.
- Statistics window.
- Manual task archiving and Archived Tasks window.
- Automatic archive threshold setting.
- Local JSON persistence.
- Optional iCloud key-value sync.
- macOS app icon.
- Open-source governance files and MIT license.
